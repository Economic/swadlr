# Clear the swadlr cache

Clears all cached metadata from the current R session. This includes
cached topics, indicators, measures, dimensions, and sources. Use this
function if you want to refresh metadata from the API.

## Usage

``` r
clear_swadlr_cache()
```

## Value

Invisible `NULL`.

## Examples

``` r
# \donttest{
clear_swadlr_cache()
# }
```
